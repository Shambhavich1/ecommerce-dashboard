📊 E-Commerce Sales Analysis Dashboard
-An interactive data analysis dashboard built using Python and Streamlit to explore, visualize, and extract insights from an e-commerce dataset.
-Demo Link: https://ecommerce-dashboard-8hhiphmyxttt5guh2drvwz.streamlit.app/

Project Overview: This project focuses on analyzing e-commerce sales data to uncover meaningful business insights such as:

1) Revenue trends over time
2) Customer purchasing behavior
3) Product category performance
4) Geographic sales distribution
5) Customer segmentation (High vs Low value)

The dashboard enables users to interact with the data through filters and dynamic visualizations.

🛠️Tech Stack used
• Python
• Pandas – Data manipulation
• NumPy – Numerical operations
• Matplotlib & Plotly – Data visualization
• Streamlit – To create Interactive web dashboard

⭐ Features
• Sales Trend Analysis
• Revenue Distribution (Histogram with Mean & Median)
• Customer Segmentation (High vs Low Value)
• Top Performing Categories
• State-wise Sales Analysis (Heatmap)
• Cancellation Analysis
• Interactive Filters (Date, Category, etc.)

⭐ Key Insights
1) A small percentage of customers contribute to a large portion of revenue (Pareto principle)
2) Strong positive relationship between number of orders and total spending
3) Certain product categories dominate overall sales
4) Sales are concentrated in specific geographic regions
5) Large number of low-value customers present growth opportunities
